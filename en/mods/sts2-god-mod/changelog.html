Changelog ​

Player-facing release notes for STS2 God Mod. The project page currently shows version 1.7.1-beta; the notes here are synced from the mod's in-code update notes through v1.7.1-beta.

Latest download: Quark download

v1.7.1-beta ​

  • Edited card damage, block, and magic values are preserved more reliably across the next combat, upgrades, downgrades, and enchantment changes.
  • Cards with permanent growth, such as The Scythe and Genetic Algorithm, now keep God Mod stat edits instead of snapping back to their original values.

v1.7.0 ​

  • Statuses can now be marked to apply automatically to yourself at the start of each single-player combat.
  • The card stat editor can now adjust a card's star cost, alongside energy cost, damage, block, and magic values. Imbued enchantment also works more reliably on supported cards.
  • Edited card keywords and added effects are preserved more reliably when the deck is saved or restored.
  • Custom deck presets refresh from local storage before use, so preset lists stay in sync after external changes.

v1.6.6 ​

  • Card added effects now support extra discard, stars, Weak, Vulnerable, Poison, Strength, and Dexterity.
  • After restarting a campfire room, the extra remove-card option becomes available again when the removed card has been restored by the room reload.

v1.6.5 ​

  • The God (G) panel now opens correctly on the latest beta build while remaining compatible with the 0.103 stable build.
  • Enemy previews in the spawn tool now handle both stable and beta bestiary layouts.
  • Card catalog search no longer triggers dynamic card-text formatting errors from cards that need combat-only values.

v1.6.4 ​

  • You can now long-press V to toggle upgraded-card previews in more card screens, including post-combat rewards, deck view, the compendium card library, smith selection, inspect-card popups, and more. Leaving the current screen automatically resets the preview.
  • The upgraded-card preview hotkey can now be rebound locally, and its long-press trigger duration can be adjusted in Other Settings.
  • Card, relic, potion, event, status, and enemy searches can now include effect text through a search toggle when you want broader matches beyond names and IDs.
  • Simple word-form normalization also helps searches like block / blocks match more reliably in the current game language.
  • The Other Settings page is more compact and easier to scan, with more related options grouped side by side.

v1.6.3 ​

  • Card and potion managers now support the same drag workflow as relics: drag catalog items into the current list to add them, or drag current items to the remove box to delete them.
  • Potion/relic quick-action hints no longer mention card-only double-click upgrades, and the deck preset save button now uses the correct label.

v1.6.2 ​

  • In same-version multiplayer sessions, God Mod can now target yourself or a teammate.
  • Supported teammate-target actions include editing resources, cards, potions, and relics; upgrading or removing target cards; removing target potions/relics; and adding preset potions/relics.
  • The target selector is hidden in singleplayer, mixed-install sessions, and sessions that have not confirmed all players are on the same God Mod version.
  • The combat shortcuts area is more compact and easier to scan.

v1.6.1 ​

  • Stable build compatibility fix.

v1.6.0 ​

  • Post-combat card rewards: customize how many choices appear.
  • Post-combat relic rewards: can become an N-pick-1 from multiple candidates.
  • Resource editor is more compact: common resource fields, reward counts, campfire helpers, and Ancient-related settings are grouped to save space.
  • Fixed linked Ancient-style reward chains sometimes not finishing cleanup after you claim a reward.
  • Fixed the God (G) panel not being hideable in some cases.
  • Seed / RNG management (beta): includes card previews, upcoming event previews, and more.

v1.5.6 ​

  • Faster list editing: double-click a current card to upgrade it, triple-click current cards/relics/potions to remove them, and triple-click catalog items to add them.
  • Fixed a stable-build issue where enabling Ancient relic refresh could leave the Neow room on a black screen.
  • Improved beta-build compatibility around the Neow room, Ancient rooms, and the G panel.
  • Added a +10 block combat shortcut and a current block field in resource editing.
  • You can now raise the maximum hand size above 10.
  • Resource inputs and buttons are more compact, with a two-column layout on wide panels and one-column fallback on narrow panels.

v1.5.5 ​

  • The G panel now opens correctly on the beta build.
  • Ancient/Neow option refreshes avoid invalid relic pairings.
  • Fixed cases where the first Neow room could stop before the next choices appeared.

v1.5.4 ​

  • Added a new Expert Mode for mixed-install multiplayer, so higher-risk testing or setup changes can temporarily lift God Mod's mixed-install safety restrictions when needed.
  • To reduce accidental desync risk, Expert Mode automatically resets to off whenever a new run starts.

v1.5.3 ​

  • Floating G button recovery is more reliable.
  • Fixed cases where the floating G button could disappear after restarting when edge snap, very low opacity, or visibility toggles were combined.

v1.5.2 ​

  • Fixed a black screen caused by relic info lookup failure on first run start.
  • Ancient event handling is more reliable.

v1.5.1 ​

  • More flexible deck management outside combat in mixed-install mode.
  • During combat, mixed-install compatibility still limits card editing to Hand for safer synchronization.
  • Outside combat, you can switch the pile selector back to Deck to inspect and organize your deck cards more easily.

v1.5.0 ​

  • Added 6 more UI language options: Traditional Chinese, Japanese, Korean, Russian, Spanish, and Brazilian Portuguese.
  • Mixed-install multiplayer now also supports +5 Energy, +5 Stars, Draw Cards, Full Heal, and Kill All Enemies.
  • While mixed-install compatibility mode is active, card management is limited to Hand and the pile selector is forced to Hand.
  • You can now add hand cards, remove hand cards, upgrade hand cards, and apply hand-card enchantments.

v1.4.3 ​

  • More stable mixed-install multiplayer.
  • Mixed-install multiplayer is now safer and more stable when players are not all on the same God Mod version, with clearer compatibility messaging in the lobby.

v1.4.2 ​

  • Added a reset entry in Other Settings and Mod Config to restore local God Mod settings back to defaults.
  • The reset flow now includes a second confirmation to help prevent accidental deletion of local settings and custom presets.
  • You can now rebind the panel toggle hotkey and the room-restart hotkey in Other Settings, and adjust their long-press trigger durations separately.
  • To avoid conflicts, these two long-press actions must use different keys.

v1.4.1 ​

  • Fixed the multiplayer campfire extra-remove flow so it is generated through the native rest-site option list, uses the correct owner player, and no longer consumes the entire campfire turn after removing a card.
  • Fixed missing icon and localization follow-up issues that could break the rest-site sync flow, prevent the remove button from updating correctly, or lead to downstream multiplayer desyncs after leaving the campfire.
  • Common numeric inputs in the G panel, such as draw count and enemy HP, now remember the last value you entered locally instead of resetting to 1 every time.

v1.4.0 ​

  • Multiplayer safety: most gameplay-editing tools that change cards, decks, relics, potions, resources, combat stats, enemies, and powers now use synchronized multiplayer actions instead of changing only the local client.
  • Multiplayer compatibility: added a Relax multiplayer lobby check toggle in Other Settings. It defaults to the safer gameplay-affecting mode. Turning it off may make mixed host/client setups easier to join, but using gameplay-editing tools in that state can still cause desyncs, disconnects, or crashes.
  • Hold-R room restart: added an Other Settings option to restart the current room by holding R for 0.3s. It is enabled by default and does not add an extra button.
  • Stability and diagnostics: improved Hold-R restart reliability in event rooms, improved infinite preset restore fallback behavior, and refined log output to make multiplayer troubleshooting easier.

v1.3.3 ​

  • Mobile client compatibility hotfix.

v1.3.2 ​

  • Added a show/hide toggle plus size and opacity customization for the floating G button, with persistence and runtime sync.
  • Fixed cases like the Architect event where an embedded combat scene could create two G buttons.
  • Added Energy +5 and Stardust +5.
  • Added Traveler mode.
  • The character filter in Add Card now reads the live character card pools, so modded characters and their cards also show up there.
  • In true multiplayer, AddCard(...) now goes through a custom synchronized action instead of changing local state directly.
  • The event catalog now includes shared ancient, so shared ancient events like Darv can also appear in the list.
  • Fixed relic / potion presets saved as new presets getting lost after restarting the game.
  • When saving relic, potion, or custom deck presets as new presets, duplicate names now automatically get -1, -2, and so on.

v1.3.1 ​

  • Fixed an issue where campfire options could become unclickable on some devices unless the run was advanced through event jumping.
  • Fixed an ancient-event flow issue that could leave the room stuck and unable to progress normally.
  • Reduced redundant UI rebuilds when opening Mod Config to improve responsiveness.

v1.3.0 ​

  • Added a God Mod settings page in Mod Config so high-risk features can be disabled directly from the settings menu.
  • Shared settings are now kept in sync between Mod Config and the in-run G panel.
  • Improved relic preset editing.

v1.2.1 ​

  • On fresh installs, Ancient Refresh and Ancient Option Override are now disabled by default, so the mod stays closer to the vanilla experience until you turn them on manually.
  • Fixed the Ancient-room refresh button visibility: it now hides correctly after you choose an option or leave the room.
  • Refined the visuals and interaction details for the settings panel and the floating G button, making controls clearer and easier to use.

v1.2.0 ​

  • Added mod language switching: Follow Game, English, and Simplified Chinese.
  • Added relic presets and potion presets, including save, overwrite, delete, and one-click apply for default and custom presets.
  • Added recent-first sorting across multiple catalogs: cards, relics, potions, statuses, events, and enemies.
  • Improved card / status / event panel workflows, including upgraded card add, card catalog preview, status preview, and better recent item surfacing.
  • Improved potion management: remove the selected potion and load current potions directly into the preset editor.
  • Continued polish for the floating G button and main panel interaction: larger panel layout, more stable drag behavior, click sound feedback for the close button, and more reliable button position restore logic.
  • Expanded logs and troubleshooting info: startup logs include game version and enabled mod list; copied logs include mod / game version, language, platform, and enabled mod details; low-value log noise was reduced.
  • Cleaned up and improved ancient event refresh logic and related logs.
  • Fix: removed unnecessary temporary debug file output.

v1.1.0 ​

  • Added the option to unlock all functions.
  • Added the "Ancestors SL" function.
  • Fixed some functions that were not working on mobile devices, such as Ancestor Refresh, Card Deletion from Fire Pit, and Setting Maximum Health Value.
  • Added support for viewing and copying Mod error logs, making it easier to troubleshoot issues.

v1.0.1 ​

  • Fixed a mod compatibility issue on mobile clients.

v1.0.0 ​

  • Completely overhauled the God Mode panel interaction, with clearer section grouping, a larger content area, and more focused access to commonly used tools.
  • Greatly expanded the card editor with multi-card checkbox selection, batch upgrade/downgrade actions, one-click upgrade for matching cards, and full-pile upgrade/downgrade support.
  • Card enchantments are no longer add-only; existing enchantments can now be edited or overwritten directly.
  • Improved card filtering with support for all characters, the current character, individual character groups, and colorless cards, making card search much more convenient.
  • Deck presets have been separated from the card editor into their own standalone section for a cleaner structure.
  • Relic editing now supports selecting multiple targets for batch add and batch remove operations.
  • Added support for extra card effects, along with a runtime patch to ensure those special effects actually take effect in play.
  • The floating G button can now be dragged freely; it only auto-snaps and hides when moved back to the screen edge, and its position is remembered.
  • Added a new About / Guide section with usage tips.
  • Enemy preview, card preview, creature drag movement, and several other interaction details have also been continuously improved.

v0.2.0 ​

  • Feature enhancement.

v0.1.0 ​

  • Initial release.
