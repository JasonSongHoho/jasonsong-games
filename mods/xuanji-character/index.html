玄羁 人物 Mod ​

玄羁是一个男性《杀戮尖塔2》自定义角色 Mod。他把战斗拆成“出牌、纳编、凝核、成型”的节奏：带羁绊的牌打出后进入机匣，机匣集齐 3 张牌时凝成机核，机核上的羁绊会在后续回合持续推动牌组节奏。

当前版本：v0.2.1-beta
发布状态：Beta
作者：Jason Song
类型：自定义角色 / 羁绊构筑 / 机核规划
是否影响玩法：是，会加入新角色、专属卡牌、专属遗物和新的构筑方向。

玩法概览 ​

玄羁不是围绕单张大牌爆发设计的角色。他的核心乐趣在于先用带标签的牌铺设方向，再通过纳编和凝核把这些标签固定成机核，最后让不同羁绊在回合开始、纳编、凝核、抽牌、消耗、战斗胜利等时机互相触发。

  • 纳编：带纳编的牌打出后进入机匣；已经纳编的牌本场战斗不会再次贡献羁绊。
  • 机匣：临时保存待凝核的牌。机匣顺序会影响下一次凝核结果，因此调度机匣本身也是决策的一部分。
  • 凝核：机匣集齐 3 张牌时凝成机核。机核提供羁绊进度，也可能带来对应的主标签身份。
  • 变阵：部分牌可以移动、移除、替换或追加标签，让你修正即将凝出的机核，而不是只能被动等待。
  • 蓄势成长：部分路线会把战斗内规划转化为金币、最大生命、牌库升级、重放或后续战斗增益。

一局游戏里，你通常不会追求所有羁绊全满。更常见的选择是先判断当前牌组需要防守、爆发、抽牌、能量、消耗、中毒/负面状态、长期成长，还是多羁绊联动，然后围绕这些方向挑牌和调整凝核顺序。

主要羁绊 ​

当前版本共有 8 个羁绊。下面只概括方向，不列出全部卡牌细节。

羁绊方向主要效果
固守防守、格挡成长固守机核会在回合开始提供格挡；羁绊进度提高后，获得格挡时追加格挡，高进度可以保留格挡并反弹受到的攻击伤害。
突袭攻击节奏、爆发突袭机核强化每回合第一次攻击；羁绊进度提高后获得活力、临时力量，并能生成额外攻击牌。
推演抽牌、手牌过滤推演机核在纳编时补牌；羁绊进度提高后提供回合开始抽牌、纳编后抽牌、弃牌/消耗选择，最终可把抽牌转化为群体伤害。
超载能量、蓄能爆发超载机核在凝核后提供能量；高进度后获得额外能量、保留未用能量，并能把当前能量转化为全体伤害。
蓄势长期收益、战斗外成长蓄势机核在未凝核的回合提供金币；高进度后胜利时提高最大生命、升级牌库或添加重放，并通过蓄势纹章带来后续战斗加成。
转化消耗、资源循环转化机核让消耗转化为抽牌；羁绊进度提高后，纳编和消耗会带来抽牌、能量与 0 费消耗随机牌。
蚀刻中毒、虚弱、易伤蚀刻机核在回合开始给敌人叠加中毒；高进度后追加虚弱、易伤，覆盖更多敌人，并在终局阶段提供大量中毒。
协同多羁绊触发器协同机核在其他主机核触发时提供格挡；高进度后会在其他羁绊触发时给格挡、抽牌、能量或伤害，并提高整体羁绊进度。

构筑方向 ​

玄羁的构筑更像“规划一套运行系统”而不是只找一张核心牌。常见方向包括：

  • 用 固守 + 推演 稳定防守、抽牌和回合质量。
  • 用 突袭 + 超载 把能量和首攻加成转成快速击杀。
  • 用 转化 + 推演 围绕抽牌、消耗和 0 费资源滚动。
  • 用 蚀刻 + 蓄势 走持续压制和长期成长。
  • 用 协同 串起多个已成型羁绊，让每次触发都带来额外收益。

兼容说明 ​

  • 需要安装 BaseLib。
  • 当前版本按本地可用的《杀戮尖塔2》v0.103.2 稳定分支和 v0.105.x beta 分支 API 做过兼容处理。
  • 这是 Beta 版本，数值、卡池和 UI 仍可能根据反馈继续调整。
