STS2 Map Enhance Mod ​

Map Enhance Mod 是一个面向 杀戮尖塔2 地图界面的可视化增强 Mod。它通过房间类型上色、路线分析、实时预览和快速开关，帮助玩家更快读懂地图、比较路线，并减少反复停下来辨认节点的成本。

当前版本：0.7.1
作者：Jason Song
类型：地图可读性 / 路线规划 / UI 辅助
是否影响玩法：否，主要改变地图信息呈现方式。

截图 ​

核心功能 ​

  • 按房间类型给地图节点上色。
  • 在地图界面显示路线分析面板，汇总每条可行路线的关键节点数量。
  • 鼠标悬停或点击路线条目时，高亮整条路线。
  • 鼠标悬浮未进入的地图房间时，预览下一步房间以及后续问号、战斗和宝箱的可能内容。
  • 支持在游戏内调整每类房间的颜色、透明度和亮度。
  • 设置页提供实时预览，不需要反复重启游戏看效果。
  • 地图打开时可实时应用配置。
  • 使用 F8 快速开启或关闭地图上色。
  • 一键恢复推荐默认配色。

支持的房间类型 ​

当前会处理这些地图节点：

类型作用
Enemy普通敌人
Elite精英
Rest休息点
Shop商店
Treasure宝箱
Unknown未知房间

Boss 和 Ancient / Special 节点目前保持原版表现，避免和原游戏的悬停、动画逻辑产生冲突。

路线分析 ​

路线分析功能会基于游戏真实的地图图结构枚举当前合法的未来路线，而不是根据屏幕坐标猜测路径。它会在地图右下角显示路线列表，并提供轻量级评分和原因说明。

每条路线会汇总：

  • 精英数量
  • 商店数量
  • 休息点数量
  • 未知房间数量
  • 宝箱数量
  • 普通怪数量

评分会参考当前生命值、金币、商店时机、精英密度、休息点缓冲等因素。它不是“最优路线计算器”，更适合当作路线比较和风险提示工具。

游戏内配置 ​

打开路径：

Settings -> 地图配置 / Map Config

你可以在这里：

  • 开启或关闭地图上色。
  • 开启或关闭图标染色。
  • 调整每种房间类型的颜色。
  • 调整透明度。
  • 调整亮度。
  • 立即预览配置效果。
  • 恢复推荐默认配色。

配置会自动保存，并在地图打开时自动应用。

默认配色思路 ​

内置默认配色优先考虑“扫一眼就能分辨”：

类型默认识别方向
Enemy柔和低饱和钢色
Elite深红色
Rest火焰红
Shop橄榄金
Treasure金紫色
Unknown绿色

这些只是推荐值，所有颜色都可以在游戏内自定义。

安装说明 ​

发布包通常包含：

  • sts2_map_mod.dll
  • sts2_map_mod.pck
  • sts2_map_mod.json
  • map_color_config.json.txt

注意：不要把 map_color_config.json 直接放进游戏 Mod 文件夹。游戏可能会把所有 *.json 当作 Mod manifest 读取，所以配置文件使用 map_color_config.json.txt。
