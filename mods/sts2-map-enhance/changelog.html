更新记录 ​

这里整理 Map Enhance Mod 面向玩家的更新说明。当前站点详情页显示版本为 0.7.1，以下已按 Mod 代码中的版本更新介绍整理到 v0.7.1。

下载最新版：夸克下载

v0.7.1 ​

  • 提升与 BaseLib 同时安装时的多人兼容性，让原版客户端更稳定地加入房间。
  • 锁定路线会在中途改走其他路线后继续保持原本高亮。

v0.7.0 ​

  • 新增地图房间预测：悬浮未进入的房间时，可以预览下一步房间，以及后续问号、战斗和宝箱的可能内容。

v0.6.3 ​

  • 新增路线锁定：选中的路线会持续高亮，切换路线偏好时也会保留。
  • 支持在路线面板中四连击某条路线，快捷锁定或取消锁定。

v0.6.2 ​

  • 修复 beta v0.105.0 下 Map Enhance Mod 可能无法加载的问题。

v0.6.1 ​

  • 修复了 v0.6.0 引入的联机混装同步问题。

v0.6.0 ​

  • 支持自定义地图画笔颜色，并支持同版本 Mod 的多人颜色同步。
  • 地图页打开牌组 / 卡牌列表时，自动隐藏悬浮小地图和路线推荐面板，返回地图后恢复。

v0.5.0 ​

  • 新增可拖拽、会记住位置的小地图。
  • 优化地图界面性能。
  • 存储选择的路线，SL 后可以恢复之前选择的路线。

v0.4.0 ​

  • 新增 6 种界面语言支持：日语、韩语、繁体中文、俄语、西语、葡语（巴西）。

v0.3.2 ​

  • 优化常驻地图 UI 逻辑带来的性能开销。
  • 提升战斗结束返回地图后路线分析面板的恢复稳定性。

v0.3.1 ​

  • 优化了路线分析面板的悬浮交互体验。
  • 修复了一些已知问题，并提升整体稳定性。

v0.3.0 ​

  • 补强路线风格预设系统，新增可随预设保存的高级评分参数。
  • 完善评分细节说明，让高级参数含义更直观。

v0.2.2 ​

  • 修复已锁定路线在重新打开地图后不再显示高亮的问题。
  • 修复战斗结束返回地图后，已锁定路线需要重新点击才会恢复高亮的问题。
  • 修复“必过商店”在手动取消后，后续地图刷新或战斗返回时又被自动勾回的问题。

v0.2.1 ​

  • 修复了当前测试版中启动过程中的兼容性问题。

v0.2.0 ​

  • 新增路线分析与推荐面板，可显示候选路线并进行评分排序。
  • 支持按精英、敌人、火堆、问号等维度切换路线排序方式。
  • 新增快捷风格预设，方便快速切换不同路线偏好。
  • 优化地图浮层 UI 与图例联动，减少信息遮挡。
  • 支持路线面板折叠、拖拽与尺寸调整，提升使用灵活性。
  • 完善设置页文案与交互，补充更多中文支持。
  • 增强日志记录与错误排查能力，方便定位问题。
  • 修复多项状态同步、界面刷新和细节表现问题，提升稳定性。

v0.1.0 ​

  • 首次发布地图增强模组。
  • 新增按房间类型给地图节点上色的功能。
  • 支持设置页实时预览地图配色效果。
  • 支持透明度与亮度调整，方便自定义显示风格。
  • 加入基础配置校验，降低错误配置带来的问题。
  • 加入日志输出，方便排查 Mod 运行问题。
  • 提供 F8 快速开关，可在游戏内快速启用或关闭 Mod。
