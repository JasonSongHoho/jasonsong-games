杀戮尖塔2 ​

STS2 God Mod ​

当前版本：1.6.0
定位：沙盒工具 / 测试工具 / 作弊型 Mod
影响玩法：是，会主动修改游戏状态。

God Mod 用于测试、沙盒玩法、构筑实验和快速起局。它可以编辑资源、卡牌、遗物、药水和战斗状态，也支持卡组预设、事件跳转、火堆辅助和多人测试相关能力。

STS2 Map Enhance Mod ​

当前版本：0.8.0
定位：地图可读性 / 路线规划 / UI 辅助
影响玩法：否，主要改变地图信息呈现方式。

Map Enhance Mod 通过房间类型上色、路线分析、双击房间筛选、房间预测、小地图、实时预览，以及 F8 快速开关，帮助玩家更快看清路线和关键节点。

FanTu / 凡途 人物 Mod ​

发布状态：预告 / 早期可玩
当前版本：v0.1.0
定位：自定义角色 / 随机牌包 / 多流派构筑
影响玩法：是，会加入新角色、专属卡牌、牌包、遗物、药水和新的构筑规则。

FanTu（凡途）是一个受《杀戮尖塔1》Packmaster 启发的《杀戮尖塔2》自定义角色 Mod。它会在每局开始时围绕角色基础牌包与若干流派牌包组成当局卡池，让玩家根据随机到的牌包组合规划构筑方向。

玄羁 人物 Mod ​

发布状态：Beta
当前版本：0.4.0-beta
定位：自定义角色 / 羁绊构筑 / 机核规划
影响玩法：是，会加入一个围绕纳编、凝核和羁绊规划的新角色。

玄羁是一个男性《杀戮尖塔2》自定义角色 Mod。他通过纳编牌、机匣、机核和 8 个主要羁绊，让玩家在战斗中规划凝核顺序，并把防守、爆发、抽牌、能量、消耗、负面状态、长期成长和多羁绊联动逐步串成一套牌组节奏。
