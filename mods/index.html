STS2 God Mod ​

当前版本：1.5.6
定位：沙盒工具 / 测试工具 / 作弊型 Mod
影响玩法：是，会主动修改游戏状态。

God Mod 用于测试、沙盒玩法、构筑实验和快速起局。它可以编辑资源、卡牌、遗物、药水和战斗状态，也支持卡组预设、事件跳转、火堆辅助和多人测试相关能力。

STS2 Map Enhance Mod ​

当前版本：0.6.3
定位：地图可读性 / 路线规划 / UI 辅助
影响玩法：否，主要改变地图信息呈现方式。

Map Enhance Mod 通过房间类型上色、路线分析、实时预览、透明度与亮度调节，以及 F8 快速开关，帮助玩家更快看清路线和关键节点。

玄羁 人物 Mod ​

状态：开发中
定位：自定义角色 / 羁绊构筑
影响玩法：是，会加入一个围绕羁绊规划的新角色。

玄羁是一个正在开发中的 杀戮尖塔2 自定义角色 Mod。她的核心玩法会围绕“羁绊”展开：玩家需要在战斗和构筑中逐步凑出不同羁绊，让牌组在合适的时机形成更强的联动。
