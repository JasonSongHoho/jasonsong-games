更新记录 ​

这里整理 STS2 God Mod 面向玩家的更新说明。当前站点详情页显示版本为 1.5.5，以下已按你提供的 Nexus Mods 更新记录整理到 v1.5.3。

v1.5.3 ​

  • 悬浮 G 按钮恢复更稳。
  • 修复了在同时使用贴边吸附、极低透明度或显示开关后，重启游戏时悬浮 G 按钮可能消失的问题。

v1.5.2 ​

  • 修复了首次开局选完角色后尝试获取遗物信息失败，导致黑屏的问题。
  • 先民事件识别更稳，减少了新版先民事件漏判导致的异常。

v1.5.1 ​

  • 混装模式下，非战斗时的牌库管理更灵活。
  • 战斗中为了更安全地同步，混装兼容模式仍然只允许操作手牌。
  • 离开战斗后，牌堆选择可以切回牌库，方便查看和整理牌库卡牌。

v1.5.0 ​

  • 新增 6 种界面语言支持：繁體中文、日本語、한국어、Русский、Español 和 Português do Brasil。
  • 混装联机兼容模式下，新增更多战斗快捷支持：能量 +5、辉星 +5、抽牌、回满血，以及秒杀全部敌人。
  • 混装兼容模式开启时，卡牌管理只允许操作手牌，牌堆选择也会强制锁定为手牌。
  • 支持添加手牌、删除手牌、升级手牌，以及对手牌附魔。

v1.4.3 ​

  • 混装联机更稳定。
  • 当房间内玩家使用的 God Mod 版本不完全一致时，联机兼容提示更清晰，混装联机流程也更稳定、更安全。

v1.4.2 ​

  • 在“其他设置”和 Mod Config 中新增重置入口，可将本地 God Mod 设置恢复为默认值。
  • 重置前会有二次确认，用于防止误删本地设置和自定义预设。
  • 可以在“其他设置”中自定义面板开关快捷键和重启房间快捷键，并分别调整长按触发时长。
  • 为避免冲突，这两个长按功能需要使用不同的按键。

v1.4.1 ​

  • 修复联机下火堆额外删牌的接入方式，改为通过原生休息站选项列表生成，按正确的拥有者玩家执行，并且删牌后不再错误占用整个休息站回合。
  • 补上图标与本地化缺失导致的后续问题，避免休息站同步链路被异常打断、删牌按钮状态不更新，或离开火堆后进一步触发联机不同步。
  • G 面板里常用的数值输入框，例如抽牌数量和敌人血量，现在会记住你上次在本地填过的值，不再每次都重置成 1。

v1.4.0 ​

  • 多人安全：绝大多数会改动卡牌、卡组、遗物、药水、资源、战斗状态、敌人和力量的编辑工具，现在都会走同步多人 action，不再只改本地客户端。
  • 联机兼容性：在“其他设置”中新增“放宽联机大厅检查”开关。默认保持为更安全的 gameplay-affecting 模式。把它关掉后，房主和客户端混装时可能更容易进房，但使用会改本局或战斗状态的功能仍可能导致联机不同步、断线甚至崩溃。
  • 长按 R 重启房间：在“其他设置”中新增长按 R 键 0.3 秒重启当前房间的开关，默认开启，且不会额外显示按钮。
  • 稳定性与排查：提升事件房中长按 R 重启房间的稳定性，补强无限预设恢复时的容错逻辑，并优化日志输出，方便继续排查多人局中的问题。

v1.3.3 ​

  • 移动端兼容性热修复。

v1.3.2 ​

  • 增加悬浮 G 按钮的显示开关、大小和透明度自定义，并接入持久化与运行时同步。
  • 修复建筑师这类事件内嵌战斗场景出现两个 G 按钮的问题。
  • 战斗快捷新增能量 +5 和辉星 +5。
  • 新增 Traveler 模式。
  • “按职业添加卡牌”的职业筛选改成动态读取实际角色卡池，所以 Mod 自定义角色的卡牌也能按职业列出来。
  • AddCard(...) 在真多人模式下改成走自定义同步 action，不再直接本地裸改。
  • 事件目录补进了 shared ancient，所以像 Darv 这类共享先民事件也能出现在列表里。
  • 修复遗物 / 药水预设“另存为新预设”后重启丢失的问题。
  • 遗物、药水、卡组自定义预设在“保存为新预设”时，遇到重名现在会自动补 -1、-2 后缀。

v1.3.1 ​

  • 修复部分设备上火堆选项无法点击、必须跳转事件后才能继续的问题。
  • 修复先民事件中可能卡住、无法正常继续流程的问题。
  • 优化 Mod Config 打开时的刷新逻辑，减少重复重建 UI 带来的卡顿。

v1.3.0 ​

  • 新增位于 Mod Config 中的 God Mod 配置页，可在设置页里直接关闭高风险功能。
  • God Mod 配置与局内 G 面板中的相关开关现已保持同步。
  • 优化遗物预设编辑。

v1.2.1 ​

  • 全新安装时，先民刷新 与 先民展示改写 现在默认关闭，避免初次使用时对原版流程造成干扰。
  • 修复先民房间中刷新按钮的显示问题：在做出选择后，或离开房间后，按钮会正确隐藏。
  • 继续优化设置项与悬浮 G 按钮的视觉表现和交互体验，让控件更清晰、更易点按。

v1.2.0 ​

  • 新增 Mod 语言切换：跟随游戏、英文、简体中文。
  • 新增遗物预设与药水预设：支持保存、覆盖、删除、一键应用默认 / 自定义预设。
  • 多个目录支持“最近使用优先”排序：卡牌、遗物、药水、状态、事件、敌人。
  • 卡牌 / 状态 / 事件相关面板体验增强：添加卡牌支持“直接升级”、卡牌目录新增独立预览、状态区新增预览面板。
  • 药水管理增强：支持删除选中药水，可把当前药水装载到预设编辑器。
  • 浮动 G 按钮和主面板交互继续优化：面板更大、拖拽区域更稳定、关闭按钮补了点击音效、按钮位置恢复逻辑更稳。
  • 日志与排障信息增强：启动时记录游戏版本和已启用 Mod 列表，复制日志时附带 Mod / game 版本、语言、平台与已启用 Mod，日志噪音进一步收敛。
  • 先民事件刷新相关逻辑和日志又做了一轮整理。
  • 修复：移除了不必要的临时调试文件输出。

v1.1.0 ​

  • 新增“解锁全部”功能。
  • 新增“允许先民 SL”功能。
  • 修复部分在手机端失效的功能：如先民刷新、火堆删牌、设置最大生命值。
  • 支持查看并复制 Mod 报错日志，排查问题更方便。

v1.0.1 ​

  • 修复移动终端中的模组兼容性问题。

v1.0.0 ​

  • 全面重做 God Mode 面板交互，分组更清晰，内容区更大，常用功能更集中。
  • 卡牌编辑大幅增强，支持勾选多张卡牌、批量升级 / 降级、同名牌一键升级、整套牌一键升级 / 降级。
  • 卡牌附魔不再只能新增，现在可以直接修改和覆盖已有附魔。
  • 卡牌筛选能力增强，加入全部职业、当前职业、各职业、无色牌等分类，找牌更方便。
  • 卡组预设从卡牌编辑中独立出来，作为单独分组管理，结构更清爽。
  • 遗物编辑支持勾选多个目标，进行批量添加和批量删除。
  • 新增卡牌附加效果相关能力，并补了对应的运行时 Patch，让特殊效果能真正生效。
  • 悬浮 G 按钮支持自由拖动，只有贴边时才会自动吸附隐藏，并且会记住位置。
  • 新增“关于 / 说明”分组，补充使用提示。
  • 敌人预览、卡牌预览、拖拽移动人物等体验也做了持续优化。

v0.2.0 ​

  • 功能增强。

v0.1.0 ​

  • 首次发布。
